Projet Kasa: Projet formatif

Contexte et objectifs:

Kasa est une entreprise de location immobilière entre particuliers. L'entreprise souhaitait moderniser son site en JavaScript, avec NodeJS côté back-end et React côté front-end. Le but était de refondre entièrement l'application afin d'adopter des pratiques modernes de développement. Ma mission était de développer l'interface utilisateur avec React, de gérer la navigation avec React Router et d'assurer la responsivité grâce aux maquettes Figma fournies.

Stack technique :

  • Frontend : React, React Router, Sass
  • Backend : Node.js (utilisé côté back-end pour l'API de gestion des données)
  • Outils : Figma (pour la maquette), Git (gestion de version), VS Code (éditeur de code)

Compétences développées :

  • Mise en place de la navigation dynamique avec React Router et gestion des routes
  • Utilisation de Sass pour la gestion des styles et la création de composants réutilisables
  • Intégration d'un design responsive et respect des normes d’accessibilité

Résultats et Perspectives d'améioration:

L’interface est désormais fluide, moderne et responsive. Un grand volume d’annonces peut être gérer efficacement tout en garantissant une navigation sans latence.

Il serait possible d'améliorer les performances de rendu en introduisant des techniques de lazy loading afin de réduire le temps de chargement initial. De nouvelles fonctionnalités telles que le filtrage et la recherche pourraient être développées.

Projet Grimoire: projet formatif

Contexte - objectifs:

Développer un site de notation de livres nommé “Mon Vieux Grimoire”, qui permettra aux membres d’ajouter des ouvrages avec une note publique, le tout en s’appuyant sur une maquette fournie par une designeuse

L’objectif principal est de concevoir et mettre en œuvre la partie back-end de l’application : en assurant un serveur opérationnel en Node.js/Express pour les opérations CRUD des livres et des notes, en mettant en place une authentification sécurisée pour les membres.

Stack technique :

  • Backend : Node.js avec le framework Express.js
  • Base de données : MongoDB pour le stockage des livres, des membres et des notes
  • Authentification : gestion sécurisée des comptes membres (inscription, connexion, token JWT ou session)
  • Coopération front-back : API REST exploitable par le front-end React

Compétences développées :

Ce projet m’a permis de consolider mes compétences métier en modélisation de données (schémas MongoDB, relations intermédiaires, bonnes pratiques de stockage), de mettre en œuvre des opérations CRUD sécurisées (validations, contrôles d’accès), et de renforcer ma capacité à travailler en tandem avec un développeur front-end sur une architecture full-stack.

Résultats et perspectives d'amélioration:

Le site back-end est opérationnel en prenant en charge l’inscription et l’authentification des membres, l’ajout d’ouvrages, l’attribution de notes publiques.L'ajout de fonctionnalités de filtrage avancé et de recherche dans la base de livres (par genre, auteur...) pourront être développées.
